5 SELECT   : 검색 표현식 (컬럼, 표현식)
<BR/>1  FROM    : 검색 대상 집합 정의 (테이블, 뷰, 서브쿼리) 
<BR/>2 WHERE    : 행 제한을 위한 조건식 정의 
<BR/>3 GROUP BY : 그룹 생성을 위한 표현식 (컬럼, 표현식) 
<BR/>4 HAVING   : 그룹 제한을 위한 조건식 정의 (그룹함수를 이용한 조건식) 
<BR/>6 ORDER BY : 정렬을 위한 표현식 (컬럼, 표현식)
